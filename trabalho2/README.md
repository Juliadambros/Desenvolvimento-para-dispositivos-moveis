# Biblioteca de livros

Aplicativo Flutter desenvolvido para o trabalho da disciplina de **Desenvolvimento de Aplicações Móveis**.

O app permite **cadastrar, editar e excluir livros**, armazenando todos os dados **localmente (SQLite)**.

---

## Entidade: Livro

- Título  
- Autor  
- Ano de publicação  
- Resumo  
- Gêneros literários *(seleção múltipla)*  
- Editora  
- Status de leitura *(Lido, Lendo, Quero ler)*  
- Capa do livro *(imagem da galeria)*

---

## Funcionalidades

- Adicionar, editar e remover livros  
- Salvar dados no banco local `sqflite`  
- Selecionar múltiplos gêneros  
- Escolher status de leitura  
- Validação e mensagens de erro
